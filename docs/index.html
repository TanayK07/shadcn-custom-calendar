Features

  • Customizable color-coded date modifiers for event types
  • TypeScript support for safer development
  • Responsive design with Tailwind CSS

Installation

npm install shadcn-custom-calendar

Customization

You can override the styling and functionality by passing your custom CSS classes and properties to the classNames and modifiersClassNames props.
